# Diecaste Leave Gen

A professional, web-based automated leave note generator designed for students at Santa Cruz High School. Built with HTML, CSS, and JavaScript, it uses ES6 template literals and string manipulation to transform form data into formal, print-ready letters.

## About

Diecaste Leave Gen is a branded student productivity tool in the Diecaste portfolio. It focuses on making formal school leave applications faster, cleaner, and easier to prepare by turning simple form inputs into polished documents that are ready to preview, print, or save as a PDF.

## How It Works

This project takes raw user inputs and passes them through a formatting engine that uses JavaScript template literals to ensure consistent indentation and formal structure, solving the hassle of manual letter writing.

## Key Features

- Generates formal school leave applications from structured form data.
- Uses JavaScript string manipulation to format names, dates, reasons, and letter content.
- Supports print-ready output for physical submission or PDF export.
- Keeps the interface lightweight with HTML, CSS, and JavaScript.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- ES6 template literals
- DOM manipulation
